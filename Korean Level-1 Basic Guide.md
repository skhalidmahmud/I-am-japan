\## 2я╕ПтГг Syllable Block Structure



Korean letters join into \*\*blocks\*\*:



```

C + V       = ъ░А (ga)

C + V + C   = ъ╣Ь (kkam)

V + C       = ьХЕ (ak)

```



\*\*Bangla equivalent:\*\* ржПржХрзЗржХржЯрж╛ ржмрзНрж▓ржХ = ржПржХржЯрж╛ ржЕржХрзНрж╖рж░/syllable.
